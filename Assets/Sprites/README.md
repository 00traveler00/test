# スプライトアセット仕様書

## 概要
このゲームは「ゆるふわサイバーパンク」をテーマにした2Dゲームです。
以下の仕様に従ってスプライトを作成してください。

## スタイルガイド
- **ゆるふわ要素**: 丸みのある形状、パステルカラー、チビキャラ、かわいい表情
- **サイバーパンク要素**: ネオンカラー（ピンク、シアン、紫、緑）、回路パターン、ホログラム、LED

## プレイヤーキャラクター (64x64px推奨)

### 1. 少女 (Girl)
- パステルピンクの髪にネオンブルーのハイライト
- 未来的なパーカー（光る回路パターン）
- テックバイザーヘッドバンド（ホログラム表示）
- LEDアクセント付きスニーカー
- アニメーション: アイドル2フレーム、歩行2フレーム

### 2. 猫（人型）(Cat)
- オレンジ色の柔らかい毛並み
- ネオングリーンのトリム付きサイバーパンクジャケット
- ホログラムタグ付きデジタルカラー
- テックグローブ
- アニメーション: アイドル2フレーム（尻尾を振る）、歩行2フレーム

### 3. 少年 (Boy)
- 青いボサボサ髪
- 光るボタン付きテックベスト
- LEDストライプ付きカーゴパンツ
- ホログラムソール付きスニーカー
- 小型ドローンコンパニオン
- アニメーション: アイドル2フレーム、歩行2フレーム

### 4. 犬（人型）(Dog)
- 茶色のふわふわした毛
- ネオンパターン付きサイバーパンクバンダナ
- 光るパッチ付きタクティカルベスト
- テックブーツ
- アニメーション: アイドル2フレーム（尻尾を振る）、歩行2フレーム

## 敵キャラクター (32x32px〜64x64px)

### 1. スライム (Slime)
- かわいいブロブ型
- ネオングロー（ピンク/シアン）
- 半透明の質感
- サイズ: 32x32px

### 2. ゴーレム (Golem)
- ロボット/テックゴーレム
- サイバーパンク要素（回路、LED）
- 重厚感のあるデザイン
- サイズ: 64x64px

### 3. トカゲ (Lizard)
- テック強化されたトカゲクリーチャー
- サイバーパーツ（義眼、装甲）
- ネオンアクセント
- サイズ: 48x48px

### 4. トーテム (Totem)
- サイバーパンク神殿/タワー
- ホログラム要素
- 浮遊するパーツ
- サイズ: 64x64px

### ボス版 (各2倍サイズ)
- デカスライム: 64x64px
- デカゴーレム: 128x128px
- デカトカゲ: 96x96px
- デカトーテム: 128x128px

## UI要素

### ヘルスバー
- ネオンスタイルのバー（赤〜ピンクグラデーション）
- サイバーパンクフレーム
- サイズ: 200x20px

### エネルギーバー
- 光るエネルギーインジケーター（シアン〜青グラデーション）
- サイズ: 200x20px

### ボタン
- 未来的なボタンデザイン
- ホバー時にネオングロー
- サイズ: 120x40px

## 環境アセット

### 床タイル (128x128px)
- サイバーパンク都市の地面
- ネオングリッド、金属プレート
- シームレスタイル

### エネルギージェム (16x16px)
- 光るクリスタル
- パーティクルエフェクト用

### ボス召喚台座 (128x128px)
- ホログラム召喚プラットフォーム
- 光るルーン/回路パターン

## 一時的な代替案（プレースホルダー）
画像生成が利用できない場合、以下の方法でプレースホルダーを作成できます：

1. **Unity Primitives**: Unityの基本図形（Sprite > Square, Circle）を使用
2. **カラーコード**:
   - プレイヤー: #FF69B4 (ピンク)
   - 敵: #FF4444 (赤)
   - ピックアップ: #00FFFF (シアン)
3. **テキストラベル**: TextMeshProで名前を表示
